# 📊 Projeto: Análise de Vendas de uma Loja

Sistema de **análise exploratória de dados (EDA)** desenvolvido com **Python, Pandas e Matplotlib** para estudo de desempenho de vendas de uma loja.

O projeto permite **tratar, organizar e visualizar dados de vendas**, simulando uma análise real de negócios para identificar padrões, tendências e indicadores importantes para tomada de decisão.

---

## 🚀 Funcionalidades

✔ Análise do valor total vendido por cidade  
✔ Identificação dos dias com maior volume de vendas  
✔ Análise da quantidade vendida por hora do dia  
✔ Análise de vendas por produto  
✔ Quantidade de clientes ativos por cidade  
✔ Geração de gráficos para visualização dos dados

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com objetivo **educacional**, para praticar conceitos importantes de análise de dados, como:

* manipulação de dados com **Pandas**
* tratamento e limpeza de dados reais
* criação de novas variáveis em DataFrame
* uso de **groupby** para agregações
* análise estatística simples de dados
* visualização de dados com **Matplotlib**

---

## 🧠 Conceitos utilizados no projeto

Durante o desenvolvimento foram utilizados alguns conceitos importantes:

* conversão de tipos de dados (`astype`, `to_numeric`)
* tratamento de strings (`replace`, `str[]`)
* criação de colunas derivadas
* agrupamento e soma com `groupby`
* análise de comportamento de vendas
* interpretação de gráficos
